Úvod

Semestrální práce z předmětu Značkovací jazyky se zabývá tvorbou XML dokumentu popisujícího soubor do kterého aplikace ukládá svá nastavení. Součástí je také tvorba XSD schéma, XSL transformace do JSON a tato dokumentace. Dokument byl tvořen jako konfigurační soubor počítačové hry.

Metodika

Všechny dokumenty byly vytvořeny v programu NetBeans IDE 8.2.

Nejprve byl vytvořen hrubý XML dokument se jmény elementů a atributů. Ten byl následně konverttován do XSD pomocí webové aplikace Freeformater v designu Russian Doll (zachovává stejnou strukturu jako zadnaý XML dokument). Tento XSD dokument byl pak ručně doplněn o restrikce na datové typy a také byla upravena celková strktuaa dokumentu do finální podoby.

Finální XML bylo vytvořeno ručně.

Validita těchto dvou dokumentů byla zkontrolována pomocí webového validátoru . Zároveň bylo zkontrolováno, že XML dokument odpovídá XSD schématu.

XSL transformace do JSON byla psána ručně.

Celý projekt byl nahrán na GitHubu.

Popis transformace

JSON

Data jsou transformována do JSON dokumentu. Jsou strukturována stejně jako výchozí XML soubor. Elementy s datovým typem string jsou zapsány v úvozovkách. Číselné datové typy a boolean jsou zapsnány bez úvozovek.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo při komunikaci mezi programy. Jeho výhodou je dobrá čitelnost a šikoké možnosti filtrování a omezování dat a datovýh typů. Práce s ním je také snadná, protože k jeho editaci stačí pouze textový editor.
